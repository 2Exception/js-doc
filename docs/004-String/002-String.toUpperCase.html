1.5.2.1. String.prototype.toUpperCase()

1.5.2.1.1. 描述

toUpperCase 会将调用该方法的字符串转为大写形式，并返回。

toUpperCase 不会影响字符串本身。

1.5.2.1.2. 参数

无

1.5.2.1.3. 返回值

一个新的字符串，转换为大写形式之后的字符串。

1.5.2.1.4. 示例

var str = 'app'
var newStr = str.toUpperCase();

console.log(str);    // app
console.log(newStr); // APP
